# 炉痕地牢 · 三层远征

一款优先适配手机浏览器的单机 PVE 卡牌地牢。无需安装，打开网页即可玩。

## 当前玩法

- 3 名英雄，各有 8 张固定初始牌与独立技能
- 3 层地牢，共 15 个房间；每层 5 个节点并在末尾挑战层主
- 普通战、精英、事件、营火、遗物宝库与首领路线
- 战斗必须把怪物生命降到 0；没有回合上限，也不按场面比分提前结束
- 能量从 1 逐回合增长，最高 10 点，高费牌有完整使用窗口
- 单位阵亡和法术施放后进入弃牌堆；牌库抽空时弃牌堆自动洗回
- 10 件遗物，精英、宝库和层主提供三选一
- 手牌使用扇形布局；手牌、战场、奖励、图鉴与放大详情复用同一张卡牌模板
- 暗黑锻炉战斗背景与多层地牢路线背景均随包离线提供

“自由铸卡 / 自定义起始牌”机制已从界面和游戏流程中删除。

## 主要文件

- `index.html`：游戏全部界面
- `complete.js`：卡牌、战斗、路线、抽弃牌循环与遗物逻辑
- `complete.css` / `polish.css` / `flow.css`：基础布局、统一卡牌模板与新版 PVE 视觉覆盖
- `assets/`：卡图图集、英雄/敌方图集和两张地牢背景
- `AI_HANDOFF.md`：给后续 AI 的完整设计与技术交接
