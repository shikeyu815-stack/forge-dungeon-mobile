# 炉痕地牢 · 完整本地试玩版

本地可直接双击 `index.html` 开始游戏，无需安装。托管版本直接发布同一份纯静态游戏，可在手机浏览器中游玩。

另一名开发者或 AI 接手时，请先阅读 `AI_HANDOFF.md`，再查看 `PROJECT_MANIFEST.json` 与 `ASSET_MANIFEST.md`。根目录游戏文件是权威源，`dist/` 由无依赖构建脚本自动生成。

## 已完成内容

- 三名玩法不同的英雄：铸炉者、孢子之母、逐影者
- 首页开始游戏后进入独立英雄选择页，可预览与实际出征完全一致的初始牌组
- 每名英雄拥有一张专属一次性法术；法术施放后不占战线，并会进入本场弃牌区
- 七房间地牢路线：普通战斗、事件、营火、精英和 Boss
- 连续生命值、牌组成长、三选一卡牌奖励、遗物和本地存档
- 三线同时结算、护盾、荆棘、中毒、偷袭、穿透和遗言
- 双方单位向战线中央对撞、爆炸、回弹和伤害浮字演出
- 卡牌图鉴收录全部英雄、中立与敌方卡牌，并提供分类筛选
- 47 张已实装卡牌各自拥有固定且不同的卡图；手牌、战场、奖励和图鉴保持一致
- 手牌、战场、奖励、图鉴、铸卡预览与放大详情使用统一 5:7 卡牌组件
- 幻想卡框突出费用宝石、大幅卡图、规则文字和底角攻防宝石；长按可查看关键词说明
- 规则效果与背景短句分区显示；手牌、战场、大卡与图鉴都会显示，紧凑卡各保留一行
- 手机竖屏专用界面，也可在电脑中以手机画幅游玩
- 自由铸卡台；护盾、直伤与荆棘可分别调节 1–4 点数值，预算、牌面和战斗效果会同步变化
- 自定义卡是可选的局外起始牌；英雄选择页可决定本次是否额外携带

## 操作

点击“开始游戏”后选择英雄，可逐个预览初始牌组，确认后再开始出征。战斗时单位牌需要选择战线，法术牌选择后可直接锁定。双方行动会同时揭示；同线单位互相攻击，空线单位直接攻击英雄。长按任意卡牌可在其上方查看放大卡面和关键词说明。
